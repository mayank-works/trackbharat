# TrackBharat

## Project Vision

TrackBharat is a modern train tracking platform for Indian Railways.

The goal is to provide:

- Real-time train tracking
- Interactive route visualization
- ETA prediction
- Train search by number and name
- Modern user experience

---

## Tech Stack

Frontend:
- React
- TypeScript
- Vite
- Tailwind CSS v4

Backend:
- FastAPI
- Python

---

## Design Language

Inspired by:

- Apple Vision Pro
- Apple Maps
- Linear

Design rules:

- Dark theme
- Glassmorphism
- Subtle orange railway accents
- Large curved UI elements
- Minimalistic and premium
- Avoid cyberpunk aesthetics
- Avoid excessive glow

---

## Current Features

Frontend:
- Navbar
- Hero section
- Premium search bar
- Tracking showcase
- Railway network section

Backend:
- FastAPI server
- Station data source

---

## Development Rules

- Prefer reusable components
- Use TypeScript types
- Preserve existing architecture
- Do not rewrite working code unnecessarily
- Keep components small and maintainable
- Build production-ready solutions
